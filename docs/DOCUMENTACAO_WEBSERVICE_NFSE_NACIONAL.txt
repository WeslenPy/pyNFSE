1. INTRODUÇĂO
1.1 Objetivo
Este documento tem como objetivo apresentar a especificaçăo técnica do WebService de Nota Fiscal de Serviço Eletrônica (NFS-e), contemplando a integraçăo com o padrăo NFS-e Nacional e os novos campos relativos ŕ Reforma Tributária (IBS/CBS).
1.2 Escopo
O escopo deste manual abrange:
Estrutura do envelope SOAP para comunicaçăo;
Campos do padrăo ABRASF 2.04;
Novos campos da NFS-e Nacional (destacados ao longo do documento);
Campos para a Reforma Tributária (IBS/CBS);
Exemplos práticos de utilizaçăo.
1.3 Público-Alvo
Este documento destina-se a:
Desenvolvedores de software;
Analistas de sistemas;
Profissionais de TI responsáveis pela integraçăo;
Empresas de software de gestăo empresarial.
1.4 Referęncias Normativas
ABRASF - Modelo Conceitual e Manual de Integraçăo da NFS-e, versăo 2.04;
NFS-e Nacional - Especificaçăo Técnica do Ambiente de Dados Nacional;
Lei Complementar nş 116/2003 - Lista de Serviços;
Reforma Tributária - Emenda Constitucional nş 132/2023;
NT SE/CGNFS-e nş 004 - Reembolso, Repasse e Ressarcimento.

2. ARQUITETURA DO WEBSERVICE
2.1 Endpoint
O WebService está disponível no seguinte endereço:
URL: http://[servidor]:8080/webservice-1.0-SNAPSHOT/Nfes
Método: POST
Content-Type: text/xml; charset=UTF-8
2.2 Estrutura SOAP
A comunicaçăo com o WebService utiliza o protocolo SOAP 1.1. A estrutura básica do envelope é apresentada a seguir:
<?xml version="1.0" encoding="UTF-8"?>
<soapenv:Envelope
    xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/"
    xmlns:nfse="http://www.abrasf.org.br/ABRASF/arquivos/nfse.xsd">
    <soapenv:Header/>
    <soapenv:Body>
        <nfse:RecepcionarLoteRps>
            <header><![CDATA[...cabecalho...]]></header>
            <parameters><![CDATA[...corpo...]]></parameters>
        </nfse:RecepcionarLoteRps>
    </soapenv:Body>
</soapenv:Envelope>
IMPORTANTE: O conteúdo dos elementos header e parameters deve estar encapsulado em blocos CDATA. O elemento LoteRps deve incluir o atributo xmlns="" para remover o namespace herdado.
2.3 Operaçőes Disponíveis
Operaçăo	Descriçăo
RecepcionarLoteRps	Recebe um lote de RPS para processamento e geraçăo de NFS-e
ConsultarLoteRps	Consulta o resultado do processamento de um lote
ConsultarNfsePorRps	Consulta uma NFS-e a partir dos dados do RPS
ConsultarNfse	Consulta NFS-e por diversos critérios
ConsultarSituacaoLoteRps	Consulta a situaçăo de processamento de um lote
CancelarNfse	Solicita o cancelamento de uma NFS-e
